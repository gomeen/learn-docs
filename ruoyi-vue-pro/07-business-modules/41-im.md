## 1. 核心概念

### 1.1 IM 核心概念

| 概念 | 说明 |
|------|------|
| 会话 | 一对一聊天或群聊 |
| 消息 | 文本、图片、文件 |
| 未读数 | 每个会话的未读消息数 |
| 在线状态 | 用户是否在线 |

### 1.2 ruoyi IM 架构

```
[客户端 A] --WebSocket--> [IM Server] --WebSocket--> [客户端 B]
                            ↓
                       [Redis Pub/Sub]
                            ↓
                       [消息存储 MySQL]
```

**特点**：
- WebSocket 长连接
- 消息持久化到 MySQL
- Redis Pub/Sub 支持多节点

### 1.3 核心表

```sql
-- 消息
CREATE TABLE im_message (
    id BIGINT,
    conversation_id BIGINT,
    from_user_id BIGINT,
    to_user_id BIGINT,
    type INT,         -- 类型：文本/图片/文件
    content TEXT,
    read_status BOOLEAN,
    create_time DATETIME
);
```

## 2. 代码示例

### 2.1 WebSocket 握手

```java
@Component
@ServerEndpoint("/ws/im/{token}")
public class ImWebSocketHandler {

    @OnOpen
    public void onOpen(Session session, @PathParam("token") String token) {
        // 1. 校验 token
        Long userId = JwtUtil.parseToken(token);
        // 2. 保存连接
        SESSION_POOL.put(userId, session);
        // 3. 通知上线
        sendToFriends(userId, "我上线了");
    }

    @OnMessage
    public void onMessage(String message, Session session) {
        // 1. 解析消息
        ImMessageBO msg = JSON.parseObject(message, ImMessageBO.class);
        // 2. 持久化
        messageService.saveMessage(msg);
        // 3. 推送
        sendToUser(msg.getToUserId(), message);
    }

    @OnClose
    public void onClose(Session session) {
        // 清理连接
    }
}
```

### 2.2 消息发送

```java
public void sendMessage(ImMessageSendReqVO reqVO) {
    // 1. 保存消息
    ImMessageDO message = new ImMessageDO();
    message.setFromUserId(getLoginUserId());
    message.setToUserId(reqVO.getToUserId());
    message.setType(reqVO.getType());
    message.setContent(reqVO.getContent());
    messageMapper.insert(message);
    // 2. 推送
    if (isOnline(reqVO.getToUserId())) {
        sendToUser(reqVO.getToUserId(), message);
    }
    // 3. 增加未读数
    redisTemplate.opsForValue().increment(
        "im:unread:" + reqVO.getToUserId() + ":" + getLoginUserId());
}
```

@Component

@Slf4j

## 4. 关键要点总结

- ruoyi IM 通过 WebSocket 实现
- 消息持久化到 MySQL
- 在线状态通过 Redis 管理
- 支持一对一聊天、群聊
- 未读数独立计算
