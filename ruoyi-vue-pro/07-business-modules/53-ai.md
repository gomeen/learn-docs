## 1. 核心概念

### 1.1 Spring AI 是什么？

**Spring AI** 是 Spring 官方推出的 AI 应用开发框架，提供了统一的 API：
- 聊天补全（Chat Completion）
- Embedding
- 图像生成
- 音频转文字
- 工具调用（Function Calling）

## 2. 代码示例

### 2.1 聊天接口

```java
@Tag(name = "管理后台 - AI 聊天")
@RestController
@RequestMapping("/ai/chat")
@Validated
public class AiChatController {

    @Resource
    private ChatModel chatModel;

    @PostMapping("/send")
    public CommonResult<String> sendMessage(@Valid @RequestBody AiChatSendReqVO reqVO) {
        // 1. 构造消息
        List<Message> messages = List.of(
            new SystemMessage("你是一个助手"),
            new UserMessage(reqVO.getMessage())
        );
        // 2. 调用模型
        ChatResponse response = chatModel.call(new Prompt(messages));
        // 3. 返回结果
        return success(response.getResult().getOutput().getContent());
    }
}
```

### 2.2 流式响应（SSE）

```java
@GetMapping(value = "/stream", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
public SseEmitter streamChat(@RequestParam String message) {
    SseEmitter emitter = new SseEmitter();
    // 流式调用
    Flux<ChatResponse> flux = chatModel.stream(new Prompt(message));
    flux.subscribe(
        chunk -> {
            try {
                emitter.send(chunk.getResult().getOutput().getContent());
            } catch (IOException e) {
                emitter.completeWithError(e);
            }
        },
        emitter::completeWithError,
        emitter::complete
    );
    return emitter;
}
```
