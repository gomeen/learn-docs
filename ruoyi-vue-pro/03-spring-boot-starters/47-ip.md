# 6.8 IP 解析与地理位置

> 掌握 yudao 的 IP 解析能力，能实现访问地区统计与限制。

## 🎯 学习目标

完成本文档后，你将能够：
- 掌握 yudao IP 解析的 API
- 理解 IP 解析库（ip2region）
- 能在 yudao 中获取当前请求 IP 与地理位置
- 了解访问日志中的 IP 记录

## 📚 前置知识

- HTTP 协议（X-Forwarded-For）
- 第三方 IP 库（ip2region）

## 1. 核心概念

### 1.1 yudao IP 工具

| 工具 | 作用 |
|------|------|
| `IPUtils` | 获取当前请求 IP |
| `AreaUtils` | IP 转地理位置（省市区） |
| `Area` | 地区数据对象 |
| `AreaTypeEnum` | 国家/省/市/区 |

### 1.2 IP 解析库

yudao 用 **ip2region**（离线 IP 库）：
- 数据文件小（几 MB）
- 查询速度快（< 1ms）
- 支持 IPv4/IPv6

## 2. 代码示例

### 2.1 获取当前 IP

```java
import cn.iocoder.yudao.framework.ip.core.utils.IPUtils;

@GetMapping("/ip")
public CommonResult<String> getMyIp() {
    String ip = IPUtils.getIpAddr();  // "127.0.0.1"
    return success(ip);
}
```

### 2.2 IP 转地理位置

```java
import cn.iocoder.yudao.framework.ip.core.utils.AreaUtils;

@GetMapping("/area")
public CommonResult<String> getArea() {
    String ip = IPUtils.getIpAddr();
    Area area = AreaUtils.parse(ip);
    // Area { country: "中国", province: "北京", city: "北京" }
    return success(area.getProvince() + " " + area.getCity());
}
```

## 3. 关键要点总结

- **`IPUtils.getIpAddr()`** 获取当前请求 IP
- **`AreaUtils.parse(ip)`** IP 转地理位置
- **库**：ip2region（离线、高性能）
- **场景**：访问地区统计、限地区访问、风控
