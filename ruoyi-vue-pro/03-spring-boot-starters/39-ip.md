# 6.8 IP 解析与地理位置

> 掌握 yudao 的 IP 解析能力，能实现访问地区统计与限制。

## 🎯 学习目标

完成本文档后，你将能够：
- 掌握 yudao IP 解析的 API
- 理解 IP 解析库（ip2region）
- 能在 yudao 中获取当前请求 IP 与地理位置
- 了解访问日志中的 IP 记录

## 📚 前置知识

- HTTP 协议（X-Forwarded-For）
- 第三方 IP 库（ip2region）

## 1. 核心概念

### 1.1 yudao IP 工具

| 工具 | 作用 |
|------|------|
| `IPUtils` | 获取当前请求 IP |
| `AreaUtils` | IP 转地理位置（省市区） |
| `Area` | 地区数据对象 |
| `AreaTypeEnum` | 国家/省/市/区 |

### 1.2 IP 解析库

yudao 用 **ip2region**（离线 IP 库）：
- 数据文件小（几 MB）
- 查询速度快（< 1ms）
- 支持 IPv4/IPv6

## 2. 代码示例

### 2.1 获取当前 IP

```java
import cn.iocoder.yudao.framework.ip.core.utils.IPUtils;

@GetMapping("/ip")
public CommonResult<String> getMyIp() {
    String ip = IPUtils.getIpAddr();  // "127.0.0.1"
    return success(ip);
}
```

### 2.2 IP 转地理位置

```java
import cn.iocoder.yudao.framework.ip.core.utils.AreaUtils;

@GetMapping("/area")
public CommonResult<String> getArea() {
    String ip = IPUtils.getIpAddr();
    Area area = AreaUtils.parse(ip);
    // Area { country: "中国", province: "北京", city: "北京" }
    return success(area.getProvince() + " " + area.getCity());
}
```

## 3. ruoyi 仓库源码解读

### 3.1 IPUtils

**文件位置**：`/Users/xu/code/github/ruoyi-vue-pro/yudao-framework/yudao-spring-boot-starter-biz-ip/src/main/java/cn/iocoder/yudao/framework/ip/core/utils/IPUtils.java`
**核心代码**（节选）：

```java
public class IPUtils {
    public static final String UNKNOWN = "unknown";

    public static String getIpAddr() {
        HttpServletRequest request = WebUtils.getRequest();
        if (request == null) return UNKNOWN;
        // 按优先级尝试多个 Header
        String ip = request.getHeader("X-Forwarded-For");
        if (StrUtil.isBlank(ip) || UNKNOWN.equalsIgnoreCase(ip)) {
            ip = request.getHeader("Proxy-Client-IP");
        }
        if (StrUtil.isBlank(ip) || UNKNOWN.equalsIgnoreCase(ip)) {
            ip = request.getHeader("WL-Proxy-Client-IP");
        }
        if (StrUtil.isBlank(ip) || UNKNOWN.equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        // 多级代理时，第一个 IP 是真实 IP
        if (StrUtil.isNotBlank(ip) && ip.contains(",")) {
            ip = ip.split(",")[0].trim();
        }
        return ip;
    }
}
```

**解读**：
- 按优先级：`X-Forwarded-For` → `Proxy-Client-IP` → `WL-Proxy-Client-IP` → `getRemoteAddr`
- 处理多层代理：取第一个 IP

### 3.2 AreaUtils

**文件位置**：`/Users/xu/code/github/ruoyi-vue-pro/yudao-framework/yudao-spring-boot-starter-biz-ip/src/main/java/cn/iocoder/yudao/framework/ip/core/utils/AreaUtils.java`
**核心代码**（节选）：

```java
public class AreaUtils {
    private static final Ip2regionSearcher SEARCHER = new Ip2regionSearcher();

    public static Area parse(String ip) {
        if (StrUtil.isBlank(ip) || !IPUtil.isIPv4Address(ip)) {
            return null;
        }
        try {
            String region = SEARCHER.search(ip);
            // 解析: "中国|北京|北京|电信"
            String[] parts = region.split("\\|");
            return new Area()
                    .setCountry(parts[0])
                    .setProvince(parts[1])
                    .setCity(parts[2])
                    .setIsp(parts[3]);
        } catch (Exception e) {
            return null;
        }
    }
}
```

### 3.3 在 API 日志中记录 IP

`ApiAccessLogFilter` 中已经自动记录了 IP 和地区，开发者无需关心。

## 4. 关键要点总结

- **`IPUtils.getIpAddr()`** 获取当前请求 IP
- **`AreaUtils.parse(ip)`** IP 转地理位置
- **库**：ip2region（离线、高性能）
- **场景**：访问地区统计、限地区访问、风控
