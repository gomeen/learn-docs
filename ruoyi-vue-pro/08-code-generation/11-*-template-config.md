## 背景

生成质量取决于字段如何配置：列表、查询、表单、校验与关联。

## 需求

1. 对导入表配置：哪些字段列表可见、可查询、表单可编辑。
2. 给一个字段配置字典组件或用户选择组件。
3. 配置非空/长度校验。
4. 若有关联表，配置关联展示字段；否则写明如何配置。
5. 选择 CRUD 模板组（非树/主子）。

## 提示

- 查询字段过多会影响列表性能。
- 主键、租户字段通常不进表单。

## 验收标准

- [ ] 列表/查询/表单字段配置合理
- [ ] 至少一种特殊组件配置完成
- [ ] 校验规则配置完成
- [ ] 模板组选择正确
- [ ] 配置可保存并再次打开不丢失

## 延伸（选做）

- 尝试树表模板配置。
- 配置枚举字段生成。
