## 背景

移动端工程与 H5/小程序发布流程需要单独验证。

## 需求

1. 打开 yudao-ui-admin-uniapp（或 mall uniapp），说明目录与请求封装位置。
2. 本地运行到 H5 或微信开发者工具（环境允许时）。
3. 完成登录页到某一业务列表的点击路径追踪。
4. 记录小程序发布的主要步骤与注意点（appid、合法域名、体验版）。
5. 对比管理端 Vue3 与 uni-app 在鉴权存储上的差异。

## 提示

- 无 appid 时可只跑 H5。
- 真机调试注意 HTTP 合法域名。

## 验收标准

- [ ] 工程结构说明完成
- [ ] 本地可运行或明确阻塞原因
- [ ] 主路径页面追踪完成
- [ ] 发布步骤清单
- [ ] 与管理端鉴权差异说明

## 延伸（选做）
