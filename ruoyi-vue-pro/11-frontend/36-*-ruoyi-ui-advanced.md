## 背景

复杂后台页：流程设计器与代码生成器 UI，以及 Vben 工程的对应关系。

## 需求

1. 打开流程设计器 UI，记录与后端 BPM 模块的 API/菜单对应关系。
2. 打开代码生成器 UI，走通或阅读：选表 → 配置 → 预览/下载 的前端步骤，对应后端接口。
3. 若仓库有 Vben 版本，对比其与 vue3+element 版在目录结构、请求封装上的 3 个差异；若无，阅读文档说明差异即可。
4. 指出生成器预览代码在前端如何展示（高亮组件/只读编辑器）。
5. 标明本机可运行项与仅阅读项。

## 提示

- 流程设计器资源体积较大，注意按需加载。
- 生成器不要对生产库误操作。
- Vben 路径以 monorepo 实际目录为准。

## 验收标准

- [ ] 流程 UI 与后端模块对应关系记录
- [ ] 生成器主步骤与接口对应关系记录
- [ ] Vben 差异 3 条（或文档级说明）
- [ ] 预览展示方式说明
- [ ] 可运行边界标明

## 延伸（选做）

- 适配 Vben 版本同一接口。
- 给生成器增加一个自定义前端模板入口阅读。
