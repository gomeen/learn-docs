## 💡 与 dify 前端目录的差异

如果你已经看过 [dify 的 web/ 目录](../dify/)，dify 用的是 **Next.js + React**，本目录讲的是 **Vue3**。两者不冲突，可以作为对比学习：
- **React 思维**：函数式、Hooks、JSX
- **Vue 思维**：模板语法、Composition API、响应式系统
