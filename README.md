# learn-docs

基于 **dify** 与 **ruoyi-vue-pro** 的后端学习笔记库。知识分三层，避免「理论 / 工程 / 项目」混在一处。

## 三层结构

| 层 | 目录 | 定位 |
|----|------|------|
| **学科基础** | [`_fundamentals/`](./_fundamentals/) | Why / 机制：算法、OS、网络栈、DB 引擎、GoF、密码学 |
| **工程公共** | [`_common/`](./_common/) | 用法 / 选型：Redis、安全、部署、SQL 实用、测试、协作 |
| **项目实战** | [`dify/`](./dify/)、[`ruoyi-vue-pro/`](./ruoyi-vue-pro/) | 语言 + 框架 + 仓库源码 |

**学习顺序（默认）**：`_fundamentals` → `_common` → 选一个项目路径深入。

主题露面与 Mastery 唯一归属见 [`_common/SIGHTING.md`](./_common/SIGHTING.md)。

**差缺补漏自检**（前端补后端、会/不会）：[`CHECKLIST-understand.md`](./CHECKLIST-understand.md)。

**Dify 主路径**（阶段门禁 + 竖切，不要 01→11 通读）：[`dify/LEARNING-PLAN.md`](./dify/LEARNING-PLAN.md)。
